# Calculator

This is a Practise Project that uses various react fundamentals such as react-router, context-api etc.

- Single Page Application .<br>
- Context API used to avoid props-driling .<br>
- Title of page changed dynamically using React Helmet .<br>
- UseState and UseEffect Hooks used .<br>
- JS Animation library such as TiltJs and ParticleJs used .<br>
